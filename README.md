## **ğŸš€ Setup & Run**  

### **1. Clone the Repository**  
```bash
git clone https://github.com/yourusername/deepbaidya-portfolio.git
cd deepbaidya-portfolio
```

### **2. Install Dependencies**  
```bash
npm install
# or
yarn install
```

### **3. Run Locally**  
```bash
npm run dev
# or

```
